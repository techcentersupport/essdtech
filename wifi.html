District Wi-Fi Access & Setup

School-Owned Devices

📶 TIGERS

Managed Wi-Fi network reserved exclusively for ESSD-owned Chromebooks, laptops, and staff workstations.

  • 🔐 Password: Automatically Configured
  • 📱 Devices: District-owned devices only
  • ⚡ Access: Auto-connects on campus
Personal Devices

📶 ESSD-Public

Open Wi-Fi network for personal phones, tablets, guest devices, and BYOD laptops on campus.

  • 🔓 Password: None (Open Access)
  • 📱 Devices: Personal smartphones, tablets, laptops
  • 🔒 Requirement: Securly Certificate required for browsing

🔒 Securly SSL Certificate Installation

To access web pages securely through ESSD-Public without security warnings or blocked sites, personal devices must have the official Securly SSL certificate installed.

📥 Download Securly Certificate (.crt)

Quick Installation Instructions

  1. Connect your personal device to the ESSD-Public Wi-Fi network.
  2. Tap the Download Securly Certificate button above to download the certificate file.
  3. Open the downloaded file on your device:
    • iOS / iPhone: Go to Settings → Profile Downloaded → Install, then enable trust under Settings → General → About → Certificate Trust Settings.
    • Android: Go to Settings → Security / Encryption → Install from storage / CA Certificate and select the downloaded certificate file.
    • Windows / Mac: Double-click the file and follow the Certificate Import Wizard to place it in the Trusted Root Certification Authorities store.
  4. Once installed, open your browser to begin browsing.
