Wi-Fi Networks & Security Setup

📶 TIGERS Network

For: District-owned Chromebooks, laptops, and staff devices.

Connects automatically on managed district hardware.

📶 ESSD-Public Network

For: Personal phones, tablets, and guest devices.

Requires the Securly SSL Certificate below for full web browsing.

Security Required

Securly SSL Certificate Installation

If you are using a personal device on the ESSD-Public Wi-Fi network, you must download and trust the district Securly SSL Certificate to browse HTTPS websites without connection warnings or security blocks.

📥 Download Securly SSL Certificate (.crt)

Installation Steps by Device:

  • iPhone / iPad (iOS): Tap the download button above. Go to Settings → Profile Downloaded → Install. Once installed, go to Settings → General → About → Certificate Trust Settings and enable full trust for Securly.
  • Android: Tap the download button above. Go to Settings → Security & Privacy → More Security Settings → Encryption & Credentials → Install a Certificate → CA Certificate and select the downloaded file.
  • Mac (macOS): Download the file, double-click it to open Keychain Access, import it into the System keychain, right-click the Securly certificate, select Get Info, and set Trust to Always Trust.
  • Windows: Download the file, right-click it, select Install Certificate, choose Local Machine, and place it in the Trusted Root Certification Authorities store.
